# 物业管理系统开发文档

## 项目概述

本项目是一个完整的物业管理系统，包含微信小程序端、Web管理后台和后端服务。采用现代化的技术栈，提供全面的物业管理解决方案。

### 系统架构

```
物业管理系统
├── 微信小程序端 (uniapp/)     # 业主、保安、维修工使用
├── Web管理后台 (frontend/)    # 物业管理员使用
├── 后端服务 (backend/)        # API服务和业务逻辑
└── 数据库                     # MySQL/PostgreSQL
```

### 技术栈总览

| 模块 | 技术栈 | 说明 |
|-----|--------|------|
| 小程序端 | UniApp + Vue.js | 跨平台小程序开发 |
| Web管理端 | Vue 3 + Element Plus | 现代化管理后台 |
| 后端服务 | Spring Boot + MyBatis Plus | Java企业级开发 |
| 数据库 | MySQL 8.0+ | 关系型数据库 |

## 开发规范

### 1. 代码规范

#### 1.1 命名规范
- **Java类名**：PascalCase (如：UserController)
- **Java方法名**：camelCase (如：getUserList)
- **Java常量**：UPPER_SNAKE_CASE (如：MAX_RETRY_COUNT)
- **JavaScript变量**：camelCase (如：userInfo)
- **CSS类名**：kebab-case (如：user-info-card)
- **数据库表名**：snake_case (如：user_info)
- **数据库字段**：snake_case (如：create_time)

#### 1.2 文件命名
- **Vue组件**：PascalCase (如：UserList.vue)
- **页面文件**：kebab-case (如：user-list.vue)
- **工具文件**：camelCase (如：dateUtils.js)
- **样式文件**：kebab-case (如：user-list.scss)

#### 1.3 注释规范
```javascript
/**
 * 获取用户列表
 * @param {Object} params - 查询参数
 * @param {number} params.pageNum - 页码
 * @param {number} params.pageSize - 每页数量
 * @returns {Promise<Object>} 用户列表数据
 */
function getUserList(params) {
  // 实现逻辑
}
```

### 2. Git规范

#### 2.1 分支策略
- **master**：主分支，生产环境代码
- **develop**：开发分支，集成测试
- **feature/xxx**：功能分支
- **hotfix/xxx**：紧急修复分支
- **release/xxx**：发布分支

#### 2.2 提交信息规范
```
<type>(<scope>): <subject>

<body>

<footer>
```

**类型说明：**
- `feat`：新功能
- `fix`：修复bug
- `docs`：文档更新
- `style`：代码格式调整
- `refactor`：重构代码
- `test`：测试相关
- `chore`：构建过程或辅助工具的变动

**示例：**
```
feat(user): 添加用户管理功能

- 实现用户列表查询
- 添加用户新增/编辑功能
- 完成用户权限分配

Closes #123
```

## 常见问题

### 1. 开发环境问题

**Q: 启动后端服务时数据库连接失败？**
A: 检查以下配置：
- 数据库是否正常运行
- 连接地址、用户名、密码是否正确
- 数据库是否已创建
- 防火墙是否阻止连接

**Q: 前端启动时端口被占用？**
A: 
```bash
# 查看端口占用
netstat -ano | findstr :8080

# 修改端口
# 在vue.config.js中修改devServer.port
```

**Q: 小程序无法连接本地服务？**
A: 
- 在微信开发者工具中勾选"不校验合法域名"
- 确保后端服务正常运行
- 检查小程序中的API地址配置

### 2. 部署问题

**Q: 生产环境跨域问题？**
A: 在nginx中配置CORS或在后端添加跨域配置：
```java
@CrossOrigin(origins = "https://your-domain.com")
@RestController
public class UserController {
    // ...
}
```

**Q: 小程序审核不通过？**
A: 
- 检查是否符合微信小程序规范
- 确保所有功能正常可用
- 准备详细的功能说明文档

### 3. 性能问题

**Q: 页面加载缓慢？**
A: 
- 使用浏览器开发者工具分析性能
- 优化图片资源大小
- 启用gzip压缩
- 使用CDN加速静态资源

**Q: 数据库查询慢？**
A: 
- 添加适当的索引
- 优化SQL查询语句
- 使用Redis缓存热点数据
- 考虑分页查询
