# 智慧物业管理微信小程序

一个基于Discord风格设计的智慧物业管理微信小程序，支持业主、维修工、保安三种角色。

## 🚀 功能特性

### 🏠 业主端功能
- **首页**：快捷服务入口、房屋信息、待办事项、公告通知、社区活动
- **物业报修**：在线报修申请、进度跟踪、AI智能分类
- **费用缴纳**：水电费、物业费在线缴纳
- **投诉建议**：在线投诉和建议反馈
- **访客通行**：访客二维码生成、车位预约
- **消息中心**：统一消息管理
- **个人中心**：个人信息管理

### 🔧 维修工端功能
- **工作看板**：任务统计、绩效数据
- **工单管理**：接单、处理、完成工单流程
- **个人中心**：个人信息、工作设置

### 🛡️ 保安端功能
- **任务中心**：巡检打卡、定位记录
- **门岗管理**：访客核实、车辆登记
- **异常上报**：突发事件上报
- **值班日志**：电子交班记录

## 🎨 设计特色

- **Discord风格**：深色主题、现代化UI设计
- **响应式布局**：适配各种屏幕尺寸
- **自定义TabBar**：流畅的切换动画
- **Emoji图标**：无需图片资源，加载快速

## 🛠️ 开发指南

### 环境要求
- 微信开发者工具
- Node.js（可选，用于包管理）

### 快速开始
1. 下载微信开发者工具
2. 导入项目：选择 `uniapp` 文件夹
3. 填入小程序AppID（或使用测试号）
4. 点击编译运行

### 演示账号
项目内置了三个演示账号，可在登录页点击"演示账号"使用：

- **业主账号**: 13800000101 / 123456
