# 物业管理系统 - Web管理端

基于Vue 3 + Element Plus开发的现代化物业管理系统，采用Discord风格的深色主题设计。

## 功能特性

### 🏠 数据仪表盘
- **KPI展示**：入住率、报修完成率、当月收费率、业主满意度
- **趋势图表**：业主满意度趋势、工单类型分布、月度收费统计、入住率变化
- **快速操作**：一键跳转到各个管理模块

### 👥 用户与房产管理
- **业主管理**：注册审核、房产绑定、信息管理
- **员工管理**：账号增删改、角色权限分配（RBAC模型）
- **房产管理**：楼栋、单元、房屋的CRUD操作

### 📋 服务工单管理
- **工单列表**：全量工单查看与筛选
- **工单分配**：手动分配/改派工单
- **进度监控**：实时监控工单处理进度
- **预警系统**：超时、紧急工单预警与干预

### 💰 财务管理
- **账单管理**：账单生成与发布（支持批量）
- **收费监控**：收费状态监控、催缴通知
- **财务报表**：财务报表导出（Excel）

### 📢 社区管理
- **公告管理**：公告的发布、编辑、置顶、删除
- **活动管理**：社区活动的创建和管理

### ⚙️ 系统设置
- **操作日志**：完整的操作日志审计
- **系统配置**：系统参数配置

## 技术栈

- **前端框架**：Vue 3 (Composition API)
- **UI组件库**：Element Plus
- **状态管理**：Vuex 4
- **路由管理**：Vue Router 4
- **图表库**：ECharts + Vue-ECharts
- **HTTP客户端**：Axios
- **构建工具**：Vue CLI
- **样式预处理**：Sass/SCSS

## 设计风格

采用Discord风格的深色主题设计：
- **主色调**：#5865f2 (Discord紫)
- **背景色**：深灰色系 (#36393f, #2f3136, #40444b)
- **文字色**：白色/浅灰色系
- **组件风格**：圆角、阴影、渐变效果
- **交互反馈**：悬停动画、点击反馈

## 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0

### 安装依赖
```bash
cd frontend
npm install
```

### 开发环境运行
```bash
npm run serve
# 或
npm run dev
```

访问 http://localhost:8080

### 生产环境构建
```bash
npm run build
```

### 代码检查
```bash
npm run lint
```

## 开发说明

### 添加新页面
1. 在 `src/views/` 下创建新的页面组件
2. 在 `src/router/index.js` 中添加路由配置
3. 在侧边栏菜单中添加对应的菜单项

### 样式开发
- 使用 SCSS 预处理器
- 遵循 Discord 风格的设计规范
- 使用 `src/styles/variables.scss` 中定义的变量

### 组件开发
- 使用 Vue 3 Composition API
- 遵循 Element Plus 组件规范
- 保持组件的可复用性
