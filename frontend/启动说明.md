# 物业管理系统启动说明

## 项目概述

这是一个基于Vue 3 + Element Plus开发的现代化物业管理系统Web管理端，采用Discord风格的深色主题设计。

## 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0 或 yarn >= 1.22.0

## 快速启动

### 1. 安装依赖

```bash
# 进入项目目录
cd frontend

# 安装依赖（使用npm）
npm install

# 或者使用yarn
yarn install
```

### 2. 启动开发服务器

```bash
# 使用npm
npm run serve

# 或者使用yarn
yarn serve
```

### 3. 访问应用

打开浏览器访问：http://localhost:8080

### 4. 登录系统

- 用户名：admin
- 密码：123456

## 项目功能

### ✅ 已完成功能

1. **数据仪表盘**
   - KPI指标展示（入住率、报修完成率、收费率、满意度）
   - 多种图表展示（折线图、柱状图、饼图）
   - 快速操作入口

2. **用户管理**
   - 业主管理：注册审核、房产绑定、信息管理
   - 员工管理：账号管理、角色权限分配（RBAC）
   - 房产管理：楼栋、单元、房屋CRUD操作

3. **工单管理**
   - 工单列表：全量查看、筛选、统计
   - 工单详情：完整的工单信息展示
   - 工单分配：手动分配处理人

4. **财务管理**
   - 账单管理：账单生成、查看、状态管理
   - 收费监控（占位页面）
   - 财务报表（占位页面）

5. **社区管理**
   - 公告管理：发布、编辑、置顶、下线
   - 活动管理（占位页面）

6. **系统设置**
   - 操作日志（占位页面）
   - 系统配置（占位页面）

### 🎨 设计特色

- **Discord风格**：深色主题，现代化UI设计
- **响应式布局**：支持桌面端、平板、手机
- **动画效果**：丰富的交互动画和过渡效果
- **数据可视化**：ECharts图表集成

## 技术栈

- **前端框架**：Vue 3 (Composition API)
- **UI组件库**：Element Plus
- **状态管理**：Vuex 4
- **路由管理**：Vue Router 4
- **图表库**：ECharts + Vue-ECharts
- **HTTP客户端**：Axios
- **构建工具**：Vue CLI
- **样式预处理**：Sass/SCSS

## 开发说明

### 添加新页面

1. 在 `src/views/` 下创建新的页面组件
2. 在 `src/router/index.js` 中添加路由配置
3. 在侧边栏菜单中添加对应的菜单项

### 样式开发

- 使用 SCSS 预处理器
- 遵循 Discord 风格的设计规范
- 使用 `src/styles/variables.scss` 中定义的变量

### 主题色系

```scss
// 主色调
$primary-color: #5865f2;      // Discord紫
$primary-hover: #4752c4;
$primary-active: #3c45a5;

// 背景色
$bg-primary: #36393f;         // 深灰主背景
$bg-secondary: #2f3136;       // 侧边栏背景
$bg-tertiary: #40444b;        // 卡片背景

// 文字色
$text-primary: #ffffff;       // 主要文字
$text-secondary: #b9bbbe;     // 次要文字
$text-muted: #72767d;         // 弱化文字
```

## 注意事项

1. **数据模拟**：当前使用模拟数据，实际项目需要对接后端API
2. **图片资源**：部分图片使用占位文件，需要替换为实际图片
3. **功能完善**：部分页面为占位页面，需要进一步开发完善
4. **权限控制**：已实现基础的路由权限，可根据需要扩展
