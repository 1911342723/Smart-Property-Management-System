## 📱 响应式断点

```scss
// 超小屏幕（手机竖屏）
@media (max-width: 480px)

// 小屏幕（手机）
@media (max-width: 768px)

// 中等屏幕（平板）
@media (max-width: 1024px)

// 大屏幕（桌面）
@media (min-width: 1024px)
```

## 🎨 新增的响应式工具类

### 容器类
- `.responsive-container`: 响应式容器，自动调整内边距
- `.responsive-card`: 响应式卡片，统一的圆角和阴影

### 栅格系统
- `.responsive-grid`: 基础网格容器
- `.grid-1`, `.grid-2`, `.grid-3`, `.grid-4`: 自动响应式列数

### 按钮组
- `.responsive-button-group`: 响应式按钮组，手机端自动纵向排列

### 隐藏/显示类
- `.hide-on-mobile`: 手机端隐藏
- `.hide-on-tablet`: 平板端隐藏
- `.show-on-mobile`: 仅手机端显示

### 文字大小
- `.text-responsive`: 响应式正文字体
- `.title-responsive`: 响应式标题字体
- `.subtitle-responsive`: 响应式副标题字体

## 🔧 使用方法

### 在组件中使用响应式mixin：

```scss
<style lang="scss" scoped>
@import '@/styles/variables.scss';
@import '@/styles/responsive.scss';

.my-component {
  padding: 20px;
  
  @include mobile {
    padding: 12px;
  }
  
  @include tablet {
    padding: 16px;
  }
}
</style>
```

### 在模板中使用响应式类：

```html
<div class="responsive-container">
  <div class="responsive-grid grid-4">
    <div class="responsive-card">卡片1</div>
    <div class="responsive-card">卡片2</div>
    <div class="responsive-card">卡片3</div>
    <div class="responsive-card">卡片4</div>
  </div>
</div>
```

## 🚀 性能优化

- 使用CSS Grid和Flexbox实现响应式布局，避免JavaScript计算
- 通过CSS transform实现侧边栏动画，性能更好
- 图表容器使用固定高度，避免布局重排

## 📝 注意事项

1. 所有使用响应式mixin的组件都需要引入`responsive.scss`
2. 新建组件建议使用响应式工具类，保持一致性
3. 图表组件在移动端可能需要特殊处理，确保可读性
4. 测试时注意检查横屏和竖屏的显示效果
